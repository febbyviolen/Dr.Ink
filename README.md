# Dr.Ink

- 2022 건국대학교 HCI 수업에서 수행한 프로젝트입니다.
- Dr.Ink는 매일 마시는 물을 관리할 수 있는 '수분 섭취 관리앱' 입니다.
- 사용자가 매일 섭취하는 수분을 기록하고 관리할 수 있도록 돕습니다.
- 사용자가 물을 마시는 것을 잊지않도록 알람을 보내줍니다.
